================================================================================
                    TEST AKSES DATA MASTER SISWA
================================================================================

IKUTI LANGKAH INI DENGAN TEPAT:

LANGKAH 1: BUKA BROWSER
-----------------------
Buka browser (Chrome/Firefox/Edge)

LANGKAH 2: LOGIN
----------------
Akses: http://localhost:8000/login

Email: admin@test.com
Password: password

Klik LOGIN

LANGKAH 3: TEST ROUTE DEBUG
----------------------------
Setelah login, akses URL ini:
http://localhost:8000/test-siswa-direct

Anda akan melihat informasi:
- User: [nama]
- Email: [email]
- Role: [role]
- Email Verified: YES/NO
- Can Access Siswa: YES/NO

LANGKAH 4: SCREENSHOT
---------------------
Screenshot halaman test-siswa-direct dan kirim ke saya!

LANGKAH 5: TEST BYPASS ROUTE
-----------------------------
Di halaman test-siswa-direct, klik link "Try Bypass Route"

Jika muncul halaman Data Siswa = BERHASIL!
Jika redirect ke dashboard = ADA MASALAH!
Jika error 403 = ROLE SALAH!

LANGKAH 6: TEST ROUTE ASLI
---------------------------
Klik link "Try Access /siswa"

Jika muncul halaman Data Siswa = BERHASIL!
Jika redirect ke dashboard = MIDDLEWARE VERIFIED BERMASALAH!
Jika error 403 = ROLE SALAH!

================================================================================
JIKA EMAIL VERIFIED: NO
================================================================================

Jalankan command ini:

php artisan tinker --execute="$user = App\Models\User::where('email', 'admin@test.com')->first(); $user->email_verified_at = now(); $user->save(); echo 'Email verified!';"

Kemudian logout dan login ulang.

================================================================================
JIKA ROLE BUKAN ADMIN
================================================================================

Jalankan command ini:

php artisan tinker --execute="$user = App\Models\User::where('email', 'admin@test.com')->first(); $user->role = 'admin'; $user->save(); echo 'Role updated!';"

Kemudian logout dan login ulang.

================================================================================
HASIL YANG DIHARAPKAN
================================================================================

Di halaman test-siswa-direct harus muncul:
- Email Verified: YES
- Can Access Siswa: YES

Jika sudah YES semua, klik "Try Access /siswa" harus bisa masuk!
